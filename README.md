## 🌟 Overview

A modern, highly-polished, and professional Curriculum Vitae (CV) built with pure web technologies. This project showcases the professional profile of **IHSAN**, an Automation Engineer, Network Technician, and Security Researcher.

It features a **one-click bilingual toggle** (English & Indonesian) and is fully optimized for digital viewing and high-quality PDF printing.

## 🚀 Key Features

- **🌐 Dual Language Support:** Instant toggle between 🇬🇧 English and 🇮🇩 Bahasa Indonesia without page reloads.
- **📸 Profile Integration:** Professional profile picture support with fallback handling.
- **📱 Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, Desktop).
- **🖨️ Print-Ready:** Custom CSS media queries specifically designed for A4 paper output and "Save as PDF" functionality.
- **⚡ Performance:** Ultra-lightweight, built with vanilla HTML/CSS/JS (no heavy frameworks).
- **🎨 Modern Aesthetics:** Clean typography using the *Inter* font and professional iconography from *FontAwesome*.

## 🛠️ Technical Stack

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom variables (CSS Properties), Flexbox, and Grid layouts.
- **JavaScript:** Lightweight logic for language switching and UI interactions.
- **Google Fonts:** Utilizing 'Inter' for maximum readability.
- **FontAwesome:** Scalable vector icons for contact and skill sections.

## 📖 How to Use

### 1. Local Viewing
Simply open `index.html` in any modern web browser.

### 2. Changing Languages
Click the floating action button at the top-right corner labeled **🇮🇩 ID** or **🇬🇧 EN**.

### 3. Saving as PDF
- Click the **"Save as PDF / Print"** button at the bottom of the page.
- In the print dialog, set the destination to **"Save as PDF"**.
- Ensure "Background Graphics" is checked in settings for the best visual result.
